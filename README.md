# Project Overview
This Power BI dashboard analyzes IPL centuries across seasons from 2008 to 2025, providing a clear and interactive view of batting performance trends in the IPL.

## Features
- Highest individual score analysis
- Most centuries by player
- Team-wise centuries
- Venue-wise conceded centuries
- Match result comparison
- Innings-wise distribution
- Interactive slicers (Year, Month)
- Navigation between report pages

## Tools Used
- Power BI Desktop
- DAX
- Data Modeling
- Interactive Visual Design
- Excel for cleaning
